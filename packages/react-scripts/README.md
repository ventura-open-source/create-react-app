# ventura-react-scripts
Features:

[ ] Webpack 4
[ ] Multiproject

## Push new version
```
- update package.json
- npm publish
```

## Get new changes
```
- git remote add upstream https://github.com/facebook/create-react-app
- git pull upstream master*
```

## Environments Files

* .env.dev
* .env.staging
* .env.prod
* .env.test

### Multiprojects

* .env.project

### Variables examples
```
- REACT_APP_*
- VENTURA_APP_*

1. VENTURA_APP_BASE_CDN
2. REACT_APP_PROJECT_NAME
3. VENTURA_APP_PROJECT_PRIMARY_COLOR
```
